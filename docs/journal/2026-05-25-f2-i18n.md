## Ce qui a été fait

Ajout de la localisation FR/EN complète à l'interface Hoshi2Star avec un bouton de changement de langue dans la toolbar.

## Fichiers créés

- `src/locales/en.json` — strings UI en anglais (toolbar, fileTree, segmentGrid, tmPanel, qaPanel, llmModal, toasts)
- `src/locales/fr.json` — strings UI en français (mêmes clés)
- `src/lib/i18n.ts` — initialisation i18next (LanguageDetector + initReactI18next, lng par défaut = `fr`, fallback = `en`, persistance localStorage)

## Fichiers modifiés

- `src/main.tsx` — import `@/lib/i18n` ajouté avant le rendu React
- `src/App.tsx` — Toolbar et LlmConfigModal traduites ; bouton toggle langue `🇬🇧 EN` / `🇫🇷 FR` (variant ghost, entre Export et nom du projet) ; `FileTreeHeader` extrait en composant pour utiliser `useTranslation` ; toasts traduits avec interpolation
- `src/components/editor/FileTree.tsx` — texte vide traduit
- `src/components/editor/SegmentGrid.tsx` — empty states, loading, footer traduits ; `t` passé à `createSegmentColumns`, `i18n.language` dans les deps de `useMemo` pour régénérer les colonnes au changement de langue
- `src/components/editor/TMPanel.tsx` — titre, états vides traduits
- `src/components/editor/QAPanel.tsx` — titre, états vides, labels erreurs traduits ; `errorLabel` accepte `t` comme paramètre
- `src/features/editor/columns.tsx` — `SegmentColumnMeta` étend avec `t: (key: string) => string` ; en-têtes de colonnes et `StatusBadge` traduits via `useTranslation`

## Dépendances ajoutées

- `i18next` 26.2.0

## Décisions prises

- **Langue par défaut : français** — `lng: 'fr'` dans l'init i18next, conformément à la spec
- **Persistance via localStorage** — `i18next-browser-languagedetector` avec `order: ['localStorage']` ; la préférence survit aux redémarrages de l'app
- **Bouton toggle** — affiche la langue *vers laquelle* basculer (si fr → bouton "🇬🇧 EN", si en → bouton "🇫🇷 FR"), placé après Exporter dans la toolbar
- **Columns i18n** — `t` injecté dans `SegmentColumnMeta` plutôt qu'importé globalement dans columns.tsx, pour éviter un couplage implicite et permettre au `useMemo` de se retriggerer sur `i18n.language`
- **errorLabel** — fonction utilitaire non-React conservée, `t` passé en paramètre plutôt que via hook
- **`FileTreeHeader`** — header du panneau gauche extrait de `App.tsx` en composant dédié pour pouvoir appeler `useTranslation` (les composants non-React ne peuvent pas utiliser les hooks)
