## Changements majeurs depuis v0.3.1

### Nouvelles fonctionnalités

- **Translate All** (`Languages` icon) — traduit l'intégralité du projet fichier par fichier en tâche de fond (`tokio::spawn`) ; ne retraduit **pas** les segments déjà traduits (`status = 'untranslated' OR target_text = ''`) ; si un fichier est entièrement traduit, il est ignoré (`pairs.is_empty() → continue`)
- **Cooldown automatique** — après chaque fichier, compare `Instant::elapsed()` vs threshold configurable ; si dépassé, émet `h2s://llm/cooling { remainingSecs }` chaque seconde ; `CooldownBadge` (`Snowflake`) s'affiche dans la toolbar avec compte à rebours `MM:SS`
- **TranslateAllDialog** — affiche le nombre de segments non traduits + fichiers + 2 inputs (durée travail / durée pause, défauts 20 min / 3 min)
- **Export All** (`Download` icon) — vérifie la complétude avant export ; dialog bloquant si segments non traduits (count affiché) ; dialog de confirmation si tout traduit
- **`get_project_stats`** — nouvelle commande Tauri retournant `{ fileCount, totalSegments, untranslatedCount }` via une seule requête SQLite avec `?1` positional binding
- **Retry failed** — bouton "Retry N failed" dans SegmentGrid toolbar pour retranslater tous les `needs_review` en un clic
- **Glossary prompt** — dialog `AlertDialog` au premier `open_project` pour proposer extraction automatique des termes glossaire ; banner non-bloquant pendant l'extraction
- **Adaptive batch split** — `llm_translate_with_split` (Box::pin recursif) : si `MAX_RETRIES` épuisés, divise le batch en deux et retente chaque moitié indépendamment ; segment terminal → `needs_review` au lieu de bloquer
