# ADR-004 : MVP limité à RPG Maker MV/MZ

## Statut
Accepté — 2026-05-24

## Contexte
4 moteurs prévus : RPG Maker MV/MZ, VX Ace, Wolf RPG, Bakin.
Chaque moteur a un format différent et une complexité d'extraction propre.
Ressources : solo dev, horizon MVP 4 mois.

## Décision
MVP = MV/MZ uniquement.
Raisons : format JSON natif (le plus simple), marché le plus large
(90%+ des jeux fan-traduits JP récents), libs Rust existantes disponibles.

## Ordre d'intégration post-MVP
F3 : VX Ace (Ruby Marshal via marshal-rs)
F4 : Wolf RPG v1/v2 (rewolf-trans)
F5 : Wolf v3/WolfX + Bakin

## Conséquences
+ MVP livrable en 4 mois
+ Validation marché avant d'investir dans les moteurs complexes
- Utilisateurs VX Ace/Wolf doivent attendre
